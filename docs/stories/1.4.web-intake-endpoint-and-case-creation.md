## User Story

**As a** potencial cliente,  
**I want** enviar meu relato inicial pelo intake web/site,  
**so that** um lead e um caso inicial sejam criados imediatamente para analise.

## Acceptance Criteria

1. Existe `POST /api/intake/lead` para intake web/site.
2. O endpoint valida o payload usando os schemas compartilhados.
3. O endpoint persiste lead e caso inicial preservando o relato bruto.
4. A resposta confirma o recebimento sem expor detalhes internos do sistema.
