## User Story

**As a** sistema SAFETYCARE,  
**I want** criar o schema inicial e a camada de acesso aos dados de intake,  
**so that** leads, clientes, casos, jobs e auditoria possam ser persistidos de forma rastreavel.

## Acceptance Criteria

1. O banco possui tabelas iniciais para `leads`, `clients`, `cases`, `workflow_jobs` e `audit_logs`.
2. Existem migracoes executaveis para criar o schema inicial.
3. Existem repositories basicos para leitura/escrita desses agregados.
4. O acesso ao banco fica restrito ao boundary de `packages/database`.
