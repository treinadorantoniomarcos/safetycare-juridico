## User Story

**As a** equipe tecnica,  
**I want** preparar a estrutura inicial do monorepo e as configuracoes compartilhadas,  
**so that** o sistema possa evoluir em fatias sem retrabalho estrutural.

## Acceptance Criteria

1. O repositorio possui `apps/web`, `apps/worker` e `packages/*` conforme `docs/architecture.md`.
2. Existe configuracao base de `TypeScript`, lint e scripts compartilhados no monorepo.
3. O app web sobe com uma pagina canario e endpoint de health.
4. O worker possui bootstrap minimo para futuras stories.
