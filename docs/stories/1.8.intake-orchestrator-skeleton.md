## User Story

**As a** sistema SAFETYCARE,  
**I want** executar um fluxo skeleton de intake no orquestrador,  
**so that** o caso avance por estados controlados mesmo antes dos agentes completos existirem.

## Acceptance Criteria

1. Existe `runIntakeFlow(caseId)` no orquestrador.
2. O fluxo minimo cobre `capture -> consent_check -> queued_for_triage`.
3. Cada transicao atualiza o estado do caso e registra execucao.
4. O fluxo pode parar com estado controlado quando encontrar bloqueio de consentimento ou erro.
