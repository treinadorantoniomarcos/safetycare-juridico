## User Story

**As a** potencial cliente via WhatsApp,  
**I want** que minha mensagem seja recebida e normalizada pelo sistema,  
**so that** meu caso entre no mesmo fluxo de intake do canal web.

## Acceptance Criteria

1. Existe endpoint de webhook para receber eventos do WhatsApp Cloud API.
2. A assinatura do webhook e validada antes do processamento.
3. A mensagem util e normalizada para o contrato canonico de intake.
4. O fluxo reutiliza a mesma rotina de criacao de lead/caso do intake web.
